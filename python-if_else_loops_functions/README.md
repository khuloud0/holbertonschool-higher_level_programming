# Python - if/else, loops, functions

This project is part of the Holberton School Higher-Level Programming curriculum.  
The goal of this project is to practice and understand the fundamental concepts of:

- if / else statements  
- loops (while, for)  
- functions  
- basic logic and expressions  
- importing modules  
- code structure and Python syntax
