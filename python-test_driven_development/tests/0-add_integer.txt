0-add_integer
=============

This module contains the function ``add_integer`` that adds two numbers.

Example
-------

Add two integers:

>>> add_integer = __import__('0-add_integer').add_integer
>>> add_integer(1, 2)
3
>>> add_integer(100, -2)
98
>>> add_integer(2)
100
>>> add_integer(100.3, -2)
98

Raise a TypeError if a is not an integer or float:

>>> add_integer("4", 5)
a must be an integer

Raise a TypeError if b is not an integer or float:

>>> add_integer(4, "School")
b must be an integer

Raise a TypeError if a is None:

>>> add_integer(None)
a must be an integer
